Quick Reference

TOPICS

Polynomial Functions

Standard Form

Terms are ordered by Degree (highest to lowest exponent).

Degree & Leading Coefficient

Degree ($n$): Highest Exponent
Leading Coeff ($a_n$): Number attached to $x^n$

Factoring Rules

$$a^2 - b^2 = (a-b)(a+b)$$
$$a^2 \pm 2ab + b^2 = (a \pm b)^2$$

Linear Functions

Definitions

Domain: All inputs ($x$)
Range: All outputs ($y$)

Vertical Line Test

A graph is a Function if no vertical line intersects it more than once.

Equation of a Line

Slope Formula ($m$)

$$m = \frac{y_2 - y_1}{x_2 - x_1}$$

Forms

Slope-Intercept: $y = mx + b$
Point-Slope: $y - y_1 = m(x - x_1)$

Special Lines

Horizontal: $y = c$ ($m=0$)
Vertical: $x = c$ ($m=$ undef)

Geometry Concepts

Lines Relationship

Parallel: $m_1 = m_2$
Perpendicular: $m_1 \cdot m_2 = -1$

Distance Formula

$$d = \sqrt{(x_2-x_1)^2 + (y_2-y_1)^2}$$
